Developer Environment
========================

此目录介绍编译或下载等工具的使用。

编译
-----------

用户可以使用Windows平台进行编译，具体使用方法参考 `Windows环境编译 <./Developer_Environment_BL702/Windows-BL702/Quickstart_Windows_msys.html>`__

下载
-----------

用户可以使用BL自主开发的工具 ``BLFlashEnv`` 进行下载，具体使用方法参考 `BLFlashEnv <./Developer_Environment_BL702/BLFlashEnv/Burning_process.html>`__

Debug
---------

选择 ``Freedom Studio + OpenOCD`` 方式进行debug。具体调试方法参考 `Freedom Studio <./Developer_Environment_BL702/freedom_studio/freedom_studio.html>`__ 。
